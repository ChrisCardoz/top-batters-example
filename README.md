# top-batters-example

A minimal TypeScript + Lit project scaffolded for development with Vite.

## Scripts

- `dev`: Start a local dev server with hot reload.
- `build`: Type-check then build for production.
- `preview`: Preview the production build locally.

## Getting started

```bash
cd top-batters-example
npm install
npm run dev
```

Then open the printed local URL. You should see the `top-batters-example` element rendering a small list of batters and batting averages.

## Entry points

- `index.html`: Loads the custom element.
- `src/top-batters-example.ts`: LitElement component definition.

## Notes

- This project uses native ES modules and targets modern browsers.
- Adjust TypeScript config in `tsconfig.json` as needed.
